\doxysubsection{Detailed Description}
A soluçao do senario 1 vai ser usar Top-\/down Dynamic Programming with Memoization ou seja vamos usar uma soluçao recursiva que vai contruir uma tabela e cima para baixo cujo o tempo de execuçao é igual a uma soluçao Bottom-\/up Dynamic Programming a catch é que normalmente essas tabelas sao de 2 entradas. Nesta soluçao nos iremos ter que trabalhar inicialmente com uma soluçao de uma tabela de 3 entradas para compensar o facto de termos 2 constraints e mais tarde de 4 entradas para ter em conta as diferentes carrinhas
